# v0.1.0 기록 (아주 쉬운 설명)

이 문서는 0.1.0 버전에서 무엇을 만들었는지, 다시 만들려면 무엇을 하면 되는지 아주 쉬운 말로 적었어요.

## 무엇을 만들었나요?
- 집(레이아웃): 위에 헤더, 가운데 내용, 아래 푸터가 있어요.
- 방(메인 페이지): 진행 상황, 해야 할 일, 출결, 일주일 시간표를 보여줘요.
- 불빛 스위치(테마 버튼): 밝게/어둡게 바꿔요.
- 너무 긴 글은 "..."으로 예쁘게 줄여요.

## 파일 목록
- `app/layout.tsx`: 집(레이아웃) 만들기 + Providers로 테마 도우미 감싸기
- `app/page.tsx`: 메인 화면 구성, 일정 제목 말줄임 처리(truncate)
- `components/theme-toggle.tsx`: 테마(밝게/어둡게) 바꾸는 버튼
- `components/providers.tsx`: 테마를 가능하게 해주는 상자(ThemeProvider)

## 다시 만들고 싶을 때(순서)
- 1) 프로젝트 준비: Next.js 14, React 18, Tailwind 설치가 되어 있어요.
- 2) `components/providers.tsx` 만들기
  ```tsx
  "use client";
  import { ReactNode } from "react";
  import { ThemeProvider } from "next-themes";

  export function Providers({ children }: { children: ReactNode }) {
    return (
      <ThemeProvider attribute="class" defaultTheme="dark" enableSystem>
        {children}
      </ThemeProvider>
    );
  }
  ```
- 3) `app/layout.tsx`에서 Providers로 감싸기 (헤더/메인/푸터 배치)
  ```tsx
  import "./globals.css";
  import { ReactNode } from "react";
  import { Providers } from "@/components/providers";

  export const metadata = { title: "Second Semester", description: "Virtual semester study planner" };

  export default function RootLayout({ children }: { children: ReactNode }) {
    return (
      <html lang="ko" suppressHydrationWarning>
        <body>
          <Providers>
            <div className="min-h-dvh flex flex-col">
              <header className="border-b bg-background/80 backdrop-blur supports-[backdrop-filter]:bg-background/60">
                <div className="container h-14 flex items-center justify-between">
                  <span className="font-semibold">Second Semester</span>
                </div>
              </header>
              <main className="flex-1">{children}</main>
              <footer className="border-t">
                <div className="container h-14 flex items-center justify-between text-sm text-muted-foreground">
                  <span>© {new Date().getFullYear()} Second Semester</span>
                  <span>v0.1.0</span>
                </div>
              </footer>
            </div>
          </Providers>
        </body>
      </html>
    );
  }
  ```
- 4) `components/theme-toggle.tsx` 만들어서 버튼 배치
  ```tsx
  "use client";
  import { Moon, Sun } from "lucide-react";
  import { useTheme } from "next-themes";

  export function ThemeToggle() {
    const { theme, setTheme } = useTheme();
    const isDark = theme === "dark" || theme === undefined;
    return (
      <button
        className="inline-flex items-center gap-2 rounded-md border px-3 py-2 text-sm hover:bg-accent"
        onClick={() => setTheme(isDark ? "light" : "dark")}
        aria-label="Toggle theme"
      >
        {isDark ? <Sun size={16} /> : <Moon size={16} />}
        <span>{isDark ? "라이트" : "다크"}</span>
      </button>
    );
  }
  ```
- 5) `app/page.tsx`에 내용 채우기
  - 진행률/할 일/출결/주간 시간표를 보여주고, 일정 제목에 `truncate` 클래스로 "..." 처리해요.

## 주의할 점
- `app/layout.tsx`는 서버 컴포넌트여야 해요. 여기에 "use client"를 쓰면 `metadata`를 내보낼 수 없어요.
- `next-themes`는 클라이언트에서만 써요. 그래서 `Providers`("use client") 안에서 쓰도록 했어요.
- 캐시 문제로 에러가 남으면 `.next` 폴더를 지우고 다시 실행해요.

## 실행하기
- 설치: `npm install`
- 개발 서버: `npm run dev` (기본 http://localhost:3000)
