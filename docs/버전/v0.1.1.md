# v0.1.1 기록 (컴포넌트 분리 & 정리)

이번 버전은 대시보드의 UI 컴포넌트를 카드 단위로 분리하고, 페이지는 조립(배치)만 담당하도록 정리했어요.

## 핵심 변경 사항
- page.tsx 단순화: mock 데이터/로직 제거, 컴포넌트 합성 전용으로 변경
- 컴포넌트 분리
  - MainProgressCard: 학기 진행률, D-Day, 오늘 남은 투두 개수 표시
  - TodoListCard: 오늘의 투두 목록 표시(체크박스 포함)
  - AttendanceCard: 출결 현황(남은/사용한 횟수 시각화)
  - WeeklyTimelineCard: 주간 스케줄 타임라인
- 임포트/네이밍 정리
  - Header 경로 케이스 일치 (`./header`)
  - AttendanceCard 기본 export 이름 파일명과 일치
  - 불필요 아이콘 임포트 제거
- 클라이언트/서버 경계 최소화
  - page.tsx에서 "use client" 제거(서버 컴포넌트 유지)

## 폴더/파일
- `app/page.tsx`: 헤더 + 4개 카드 컴포넌트 배치(레이아웃 조립)
- `app/_components/MainProgressCard.tsx`
- `app/_components/TodoListCard.tsx`
- `app/_components/AttendanceCard.tsx`
- `app/_components/WeeklyTimelineCard.tsx`
- `app/header.tsx`: 헤더(테마 토글 포함)

## 앞으로의 권장 개선
- 카드 내부 mock 데이터 → 페이지/서버에서 props로 주입
- "use client"가 불필요한 카드들은 서버 컴포넌트로 전환
- 타입 공통화(`/types`) 및 유효성(Zod) 분리
- 간단한 테스트(RTL/Cypress) 추가

## 실행 방법
- 설치: `npm install`
- 개발 서버: `npm run dev` → http://localhost:3000
